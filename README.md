# UI library For Vue

## Prerequisites

- npm >=v6.12.0
- node >=12.13.0

## Components & Feature

- [x] Button
- [x] Icon
- [x] Input 输入框
- [x] InputNumber 计数器
- [x] Uploader 上传图片
- [x] Radio 单选框
- [x] Checkbox 复选框
- [x] Form 表单
- [x] Toast 表单
- [x] Message 消息提示
- [x] Messagebox 弹窗
- [x] Modal 复杂弹窗
- [x] 支持按需加载组件
